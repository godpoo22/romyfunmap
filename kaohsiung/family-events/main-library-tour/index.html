高雄市立圖書館總館親子活動

適合整理說故事、親子閱讀和館舍導覽，之後可用同一頁持續補新梯次。

詳細資訊

活動費用

收費方式： 依各場活動公告為準。

熱門說故事與講座可能需要提早報名或現場候補。

活動內容

適合整理： 親子說故事、閱讀推廣與館舍導覽。

安排方式： 可和總館閱覽或周邊商圈一起排 1～3 小時。

基本資訊

活動型態： 館舍活動 / 閱讀推廣。

建議停留： 1～2 小時。

雨天備案： 適合。

交通與停車

地點： 高雄市前鎮區新光路61號。

停車： 館內停車場。

適合年齡

幼兒到國小都可依主題選擇，若孩子願意坐下聽故事或參與閱讀活動，這類型活動很穩定。
